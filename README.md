# Clinical Drug Safety System

## Overview
This Python project simulates safe medication administration by evaluating patient-specific conditions and allergies. The system ensures that prescribed drugs do not conflict with patient health conditions or trigger allergic reactions.

## Objectives
- Check drug safety based on patient conditions and allergies
- Prevent unsafe medication administration
- Demonstrate basic Clinical Decision Support System (CDSS) functionality

## Features
- Store patient information (conditions, allergies, current medications)
- Evaluate drug safety with defined rules
- Alert for unsafe prescriptions
- Track administered medications

## Technologies
- Python 3

## Healthcare Relevance
Medication safety is critical in hospitals and clinics. This project demonstrates how CDSS can reduce medication errors, protect patients, and support clinical decision-making.

## Future Improvements
- Integrate with hospital EMR systems
- Add persistent database (SQLite)
- Include multiple drugs and interaction checks
